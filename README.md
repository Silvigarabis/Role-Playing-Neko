# Role-Playing-Neko

这是一个从 CSneko 的 [toNeko](https://github.com/CSneko/toNeko) 插件受到启发而创建的项目喵！
就像toNeko，这个东西也意图提供一个让玩家体验成为猫娘的插件喵！

这个程序还在编写的过程中喵！目前暂时还不能游玩喵…
